Project Title:

Student Timetable Manager (Python GUI Application)

---

1. Problem Statement

Students often have difficulty maintaining an organized class timetable, especially when schedules change frequently or when managing multiple subjects. Traditional handwritten timetables can become messy, misplaced, or require constant rewriting.
There is a need for a simple digital tool that allows students to quickly record, update, and view their daily schedule through an easy-to-use interface.

---

2. Scope of the Project

The scope of this project includes the development of a lightweight desktop application using Python.
The system provides the following capabilities:

Input of subject name

Selection of a specific day of the week

Entering start and end times in HH:MM format

Validating time inputs

Displaying all stored timetable entries

This application is intended for individual student use and is designed to be simple, fast, and user-friendly. It does not include database storage or advanced editing features in the current version.

---

3. Target Users

The primary target users are:

School students

College students

Anyone who needs a small tool to store a simple timetable

The application benefits users who prefer a small GUI-based timetable management tool without needing web access or complex installations.

---

4. High-Level Features

Tkinter-based GUI for easy interaction

Day selection menu to pick weekdays

Entry fields for subject, start time, and end time

Time validation using the Python datetime module

Error handling for empty fields or incorrect formats

Popup-based timetable display
